Welcome to My Logstash pattern file repo.

This is a project that I am trying to maintain mainly for my keeping of logstash pattern files for different types of logs that I run across in my current environment.

How to use

If you look at the ndtest.conf it will give you an idea about how to actually use pattern files.

Current pattern files

Currently there are only patterns for named query logs. I will be adding more as I create them.
